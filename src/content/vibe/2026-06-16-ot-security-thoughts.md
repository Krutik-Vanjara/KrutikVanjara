The mental model shift that unlocked OT/ICS for me: in IT, availability is nice to have. In OT, availability *is* safety. A patch that reboots a SCADA node isn't an inconvenience — it could stop a production line or worse.

This changes everything downstream: how you do risk assessment, how you segment networks, how you justify security spending to plant managers who see uptime as the only metric that matters.

IEC 62443 exists because the IT security playbook genuinely doesn't transfer.
