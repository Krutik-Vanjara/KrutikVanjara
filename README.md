# Krutik Vanjara — Personal Portfolio

**IoT & Cybersecurity Engineer · M2 ESILV Paris · Greater Paris, France**

Personal portfolio and technical blog built on [Astro Navfolio](https://github.com/dodolalorc/astro-navfolio) — a calm, writing-first static site for notes, projects, and public identity.

---

## What's here

- **Home** — who I am, what I'm building, where to find me
- **Blog** — technical deep-dives: IoT platforms, OT/ICS security, AI agents, DevOps
- **Projects** — documented engineering work and side projects
- **Vibe** — short notes on security, engineering, philosophy
- **About** — full background and technical profile

## Stack

Astro 6 · Tailwind CSS 4 · Pagefind full-text search · MDX · Vercel

## Local development

```sh
npm install --legacy-peer-deps
npm run dev          # http://localhost:4321
npm run build        # production build + search index
npm run preview      # preview production build
```
